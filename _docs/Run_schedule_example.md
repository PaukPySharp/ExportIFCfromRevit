# Пример файла с графиком запуска ExportIFC

Этот файл — пример того, как можно фиксировать расписание автозапуска скрипта ExportIFC.

Рекомендуется хранить такой txt-файл рядом с папкой `admin_data` на общем сетевом или локальном ресурсе,
где лежат файлы управления выгрузкой.  
По этому файлу можно быстро понять, как настроен Планировщик задач Windows
на компьютере, который выполняет выгрузку IFC.

Пошаговая настройка Планировщика и остальных компонентов окружения описана
в [ExportIFC_manual](ExportIFC_manual.md) и в разделе Quick start корневого
[README репозитория](../README.md); этот пример служит дополнением к ним и должен оставаться
согласованным с фактическим расписанием, указанным в этих документах.

## Пример ежедневного расписания для одного компьютера

Выгрузка IFC на ПК `IFC-SRV-01` запускается в:

- 11:00
- 16:00
- 22:00

(название ПК можно подставить своё или опустить, если компьютер один)

> Настройка расписания зависит от нагрузки на модели и инфраструктуру.
> Этот файл — лишь пример, адаптируйте под свои требования.

## Как использовать этот пример

1. Создайте рядом с `admin_data` текстовый файл, например `_График запуска программы.txt`.
2. Скопируйте в него этот пример и пропишите фактические времена запуска
   для ваших компьютеров (и при необходимости укажите их имена).
3. При изменении настроек в Планировщике задач обновляйте файл,
   чтобы график всегда соответствовал реальному расписанию.
